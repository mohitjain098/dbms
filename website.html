DATA DEFINITION LANGUAGE

Commands in SQL that are used to delete,create, manipulate the structure of database.

1. CREATE:- This command is used to create a new database or a table to store.

2. RENAME:- This command is used to change the name of existing database or table.

3. TRUNCATE:- This command used to delete all the inserted rows from the existing database or table. used to freeup the space used by table

4. ALTER:- Commands used to make changes in the columns of existing database or table. It can be used to modify , add or drop the columns.

5. DROP:- drop commands is used to delete the entire table.

DATA MANIPULATION LANGUAGE

Commands in SQL that are used to enter , delete or upload the data in the database.

1. INSERT :- Is is used to insert data in the existing database or table.

2. UPDATE :- Commands used to alter the records of existing table.

3. DELETE:- Command used to remove the previous records of the existing database or table.

CONSTRAINTS

It is used to put restrictions on the insertion of data in the database ot table.

1. PRIMARY KEY :- Constraint that is used to put restriction of unique and not null values.

2. FOREIGN KEY :- Contraint that is used to take reference from another table's primary key.

3. NOT NULL :- This constraint ensures that a column cannot have a NULL value.

4. UNIQUE :- This constraints ensures that a column should have all different values.

= EQUAL TO
<>,!= Not equal to
"<" Less than
> Greater than
<= Less than equal to

OR For the row to be selected at least one of the conditions must be true

GROUP FUNCTIONS IN SQL

These are the build-in functions in SQL that operates on groups of rows and return one value for the entire group

1 . COUNT : Returns the number of rows in table that satisfies the condition specified in the WHERE clause.

2. SUM : This function is used to get the sum of numeric column

3. MAX : This function is used to get the maximum value from a column.

4. MIN: The function is used to get the minimum from a column.

5. AVG : The function is used to get the average value of numeric (column).

SINGLE ROW FUNCTIONS

These are the functions which operates on single row and return one result per row

1. CHARACTER FUNCTIONS : Used to manipulate text strings.

Thses can be divided into two sub-functions:-

Case-manipulation functions

(i) lower(string):- This function converts mixed and uppercase strings to lowercase.

(ii) Upper(string):- This function converts mixed and lowercase strings to uppercase.

(iii) Init(string):- This function converts first letter to uppercase and remaining letters to lowercase

Character-manipulation functions

(i) Concat(string1,string2):- thus functions joins two string values together.

(ii) Substr ( string,m[,n]) :- This function returns a substring, n characters long from the string specified from the starting position number m.

(iii) length(string):- This function returns the length of a string as a numeric value.

2. NUMERIC FUNCTIONS

(i) ABS(n) :- It returns the absolute value of value passed

(ii) CEIL(n) :- It returns the smallest internet greater than or equal to n.

(iii) FLOOR(n) :- It returns the largest integer less than or equal to n

(iv) MOD(m,n) :- Returns the remainder of m divided by n.

TRANSACTION CONTROL LANGUAGE

TCL is used to manage transaction in a database. It is mainly changes made by DML statements.

1. COMMIT:- It is used to permanent save any transaction into the database.

2. ROLL BACK:- It is used to get the data or restore the data to the last save point or last commited state.

3.SAVEPOINT:- It is used to save the data at a particular point temporarily, so that whenever required it can be roll back to that particular point.

DATA CONTROL LANGUAGE

These commands are used to control privilege in the database.The privileges are required for performing all the database operations like create. View tables.

1. GRANT:- This command is used to grant permission to the user to perform a particular operation on a particular object.

2. REVOKE:- This command is used to take permission/privilege back from user.

JOINS IN SQL

SQL Joins are used to relate information in different tables. It is used to combine columns of different tables. Connection between tables is established through the WHERE clause, called SQL Join condition.


1. EQUIJOIN :- When two tables are joined together using equality of values in one or more columns, they make an equi join.

 
        
            
			
2. CARTESIAN JOIN :- If SQL Join condition is committed or if it is invalid then operation will result in a Cartesian product. It returns the number of rows equal to the product of all the tables rows that joined together.

		
 
            
            
			
3. OUTER JOIN :- The SQL Join returns all the rows from both tables which satisfy the join condition along with the rows that does not satisfy the join condition

		
 
        
            
			
4. SELF JOIN:- Self join is mainly preferred to join a table with itself. This means that each row of the table is combined with itself and with every other row of table.
